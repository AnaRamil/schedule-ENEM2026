# ENEM 2026 Editable Schedule

Um cronograma de estudos personalizável para o ENEM 2026, criado como uma página HTML simples e prática.

## Visão geral

Este repositório contém um cronograma de estudos totalmente editável em formato HTML, voltado para estudantes que desejam organizar sua rotina de preparação para as provas do ENEM 2026.

## Arquivo principal

- `schedule-ENEM.html`: página HTML com layout responsivo e área de conteúdo editável para inserir horários, tarefas e temas de estudo.

## Recursos

- Layout visual organizado por períodos do dia (manhã, tarde e estudos)
- Blocos de estudo com horários fixos e áreas editáveis
- Sessão de planejamento de redação com temas semanais sugeridos
- Botão de impressão integrado para gerar uma versão em PDF ou imprimir direto do navegador
- Design leve, sem dependências externas além da fonte do Google

## Como usar

1. Abra o arquivo `schedule-ENEM.html` no navegador.
2. Edite os campos com seu horário, matérias e objetivos.
3. Use o botão `Imprimir Cronograma` para salvar em PDF ou imprimir.

## Personalização

- Alterar horários e atividades diretamente no texto editável.
- Atualizar os temas de redação e os conteúdos de estudo conforme seu plano.
- O layout é construído com HTML e CSS simples, permitindo ajustes rápidos no código.
